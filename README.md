USK [ Universal Studios Korea ] . Stand alone type
===================================================
2020年12月から2021年7月までのJAVA研修で初めて作ったチームプロジェクトです。
1月21日から3月23日までの日程でした。
韓国にもユニバーサルスタジオができてほしいと願いの気持ちが込めてあります。



概要
----
### なぜ作ったのか。
* チケット販売所の混雑状況を回避する。
* ホームページから沢山の情報や訪れた客の感想、レビューを見る。
* ホームページ -> 情報提供 -> 予約・販売 -> 利益


使用技術
--------
* Oracle SQL
* JAVA
* eclipse
* Tortoise SVN
* Photoshop
* Illustrator
